
مرحله ۱: دانلود فایل‌های مورد نیاز
به مسیر اشتراک‌گذاری زیر دسترسی پیدا کنید:
\\share\public\فنی

فایل‌های زیر را دانلود کنید:

nssm.zip (ابزار مدیریت سرویس‌های ویندوز)

php.zip (موتور اجرای اسکریپت‌های PHP)

مرحله ۲: استخراج فایل‌ها
فایل‌های دانلود شده را در ریشه درایو C:\ استخراج کنید.

ساختار پوشه‌ها پس از استخراج باید به صورت زیر باشد:

bash
C:\nssm
C:\php
مرحله ۳: ویرایش فایل Batch
به مسیر زیر بروید:
C:\installer\files\

فایل installer.bat را با ویرایشگر متنی (مثل Notepad) باز کنید.

در خط ۶۹ این فایل، آدرس IP فعلی سیستم را پیدا و با IP واقعی سرور جایگزین کنید:
قبل از تغییر:

bash
set SYSTEM_IP=192.168.1.100
پس از تغییر (مثال):

bash
set SYSTEM_IP=10.20.30.40
فایل را ذخیره کنید.

مرحله ۴: به‌روزرسانی پایگاه داده
به پایگاه داده moadian متصل شوید.

جدول Personal را حذف کنید:

sql
DROP TABLE IF EXISTS Personal;
اسکریپت‌های SQL پیوست شده (Views و Stored Procedures) را اجرا کنید.


⚠️ هشدار: قبل از اجرا از پایگاه داده Backup بگیرید.

مرحله ۵: اجرای نصب‌کننده
فایل installer.bat را پیدا کنید (معمولاً در C:\installer\).

PowerShell را به عنوان Administrator اجرا کنید.

دستور زیر را وارد کنید تا خطمشی اجرای اسکریپت تغییر کند:

powershell
Set-ExecutionPolicy RemoteSigned -Force
اسکریپت نصب را اجرا کنید:

powershell
.\installer.ps1
نکات عیب‌یابی
خطا در اجرای اسکریپت:

مطمئن شوید PowerShell اجازه اجرا میدهد (Get-ExecutionPolicy).

مسیر فایل‌های استخراج شده را بررسی کنید.

خطای پایگاه داده:

از تطبیق ساختار جداول با اسکریپت‌های جدید اطمینان حاصل کنید.

مشکل در ویرایش فایل BAT:

از ویرایشگرهایی مثل VS Code یا Notepad++ استفاده کنید تا فرمت خطوط به هم نخورد.
